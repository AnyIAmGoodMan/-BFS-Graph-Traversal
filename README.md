# BFS Graph Traversal

Статус: Работающий алгоритмический проект.

Реализация поиска в ширину для двух типов представления графа: матрица смежности и матрица инцидентности. Проект включает генерацию случайных графов, выполнение BFS и визуализацию результатов.

## Файлы проекта

- RandomAdjacencyMatrixGenerator.py - генерация матрицы смежности
- RandomAdjacencyMatrixGeneratorWithDensity.py - генерация матрицы смежности с заданной плотностью рёбер
- RandomIncidenceMatrixGenerator.py - генерация матрицы инцидентности
- BFS_Adjency.py - обход графа в ширину по матрице смежности
- BFS_Incidence.py - обход графа в ширину по матрице инцидентности
- BFS_AdjencyVisualization.py - визуализация графа и порядка обхода для матрицы смежности

## Порядок запуска

Проект запускается в три этапа:

1. Генерация матрицы. Запустите один из генераторов. Матрица сохранится в текстовый файл.

   python algorithms/RandomAdjacencyMatrixGenerator.py

2. Запуск алгоритма BFS. Алгоритм читает матрицу из файла и выполняет обход.

   python algorithms/BFS_Adjency.py

3. Визуализация. Отображает граф и последовательность обхода вершин.

   python algorithms/BFS_AdjencyVisualization.py

## Пример использования

1. Сгенерируйте матрицу смежности с помощью RandomAdjacencyMatrixGenerator.py.
2. Запустите BFS_Adjency.py. Программа выведет порядок обхода вершин в консоль.
3. Запустите BFS_AdjencyVisualization.py. Увидите граф с подсвеченным порядком обхода.


## Требования

Для запуска проекта необходим Python 3.6 или выше и следующие библиотеки:

- numpy
- matplotlib
- networkx

Установка зависимостей:

   pip install -r requirements.txt

## Примечания

- Визуализация реализована только для матрицы смежности. Для матрицы инцидентности визуализация отсутствует из-за специфики представления.
- Матрицы сохраняются в текстовые файлы.
